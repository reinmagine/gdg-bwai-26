# GDG Build with AI 2026: Break the Pattern

This repository contains mini projects and exercises I built during the GDG Build with AI 2026: Break the Pattern event. I utilized Google Antigravity and its AI agents to accelerate and simplify the development process.

## Projects Overview

Throughout the event, I learned working on several projects to explore the capabilities of AI orchestration. The directories in this repository include:

*   **concierge-app** - An application built using Antigravity agents, demonstrating how to use the Model Context Protocol (MCP) to handle complex tasks.
*   **dynamic-website** - A technical conference website generated with Python, Flask, HTML, CSS, and JavaScript. It features an event schedule, speaker details, and search functionality.
*   **productivity-app** - A web application featuring a Pomodoro timer with a clean layout to help with focus and task management.
*   **unit-tests** - Python scripts demonstrating how to write and run unit tests for an e-commerce order system, covering edge cases and custom errors.

## Learnings

The event provided a lot of insights into the future of software development, focusing on how AI changes the way we build applications.

Topics Explored

*   **Women in Tech** - The talks highlighted the current state of women in the tech industry. With only 18% being women, there is a strong need to encourage more young students to pursue STEM fields.
*   **Agentic AI and Orchestration** - Discussions covered the shift from traditional coding, where every rule is handwritten, to orchestrating intelligence using AI. The new role of the developer involves designing with AI rather than just writing conditionals.
*   **Accessibility through AI** - Bridging the accessibility gap was a major focus. Designing with empathy is crucial, and utilizing AI can help create applications that are operable and understandable for everyone.
*   **Google Antigravity and AI Studio** - The sessions introduced Gemini 3 and Google Antigravity. We explored concepts like "Vibe Coding" and leveraging the IDE's browser agent to complete code, test functionalities, and build applications efficiently.

## Personal Reflection

Participating in GDG Build with AI 2026 was a great experience. I learned that the future of coding is moving beyond just typing out syntax or writing complex conditional statements; it is about learning how to orchestrate AI agents to build solutions. 

Using Google Antigravity allowed me to build websites, productivity tools, and tests much faster than traditional methods. It shifted my perspective from being a traditional programmer to an AI orchestrator. I also realized that empathy and accessibility remain at the core of good design, and we now have more powerful tools to bridge those gaps and build more inclusive digital experiences.
